# Claude Everything

Reusable Claude Code skills for Python/FastAPI backend projects.

## Skills

| Skill | What it covers |
|---|---|
| `coding-conventions` | Style, naming, error handling, logging, config, database, and API conventions |
| `engineering-guardrails` | 30+ anti-patterns checklist — what NOT to do |
| `layer-rules` | Per-layer responsibilities and DOs/DON'Ts (API, Services, Domain, Persistence, External Clients) |
| `migrations` | Alembic migration rules — generate, run, and configure |
| `testing-guidelines` | Test pyramid, conventions, factories, and per-layer test rules |
| `push-to-remote` | Git workflow — branch naming, squash, rebase, push, and MR creation |

## Install

```bash
npx skills add AishaJaved/claude-everything
```

## Stack

These skills are opinionated for:
- Python 3.12+
- FastAPI
- SQLAlchemy + Alembic
- Pydantic v2
- pytest + factory_boy
- GitLab (push-to-remote)
